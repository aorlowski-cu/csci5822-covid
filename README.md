# csci5822-covid

## PYMC Install instructions (for Windows)
to install pymc create a conda environment using miniconda or anaconda
https://www.pymc.io/projects/docs/en/latest/learn/core_notebooks/pymc_overview.html

### Create conda environment
**in conda prompt:**

**1**
```conda create -c conda-forge -n pymc_env "pymc>=4"```

**2**
```conda activate pymc_env```

**3**
```python your_file.py```
